## Rollback Plan

If issues arise during testing:

1. Revert pyproject.toml and Dockerfile:
```bash
git revert HEAD~2..HEAD
```

2. Rebuild with previous Python version:
```bash
uv sync --frozen
docker-compose build
```

3. All changes are isolated to version requirements, no data migration needed
